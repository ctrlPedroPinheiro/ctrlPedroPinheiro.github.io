Sobre Mim

Sou um desenvolvedor focado em construir soluções eficientes e arquiteturas escaláveis. Com uma base sólida em Java, minha experiência transita entre o desenvolvimento de software tradicional — como a criação de sistemas web com Spring Boot — e áreas matemáticas avançadas da computação.

Acredito que um bom profissional não apenas utiliza frameworks, mas entende a lógica subjacente. Por isso, aplico meus conhecimentos em Pesquisa Operacional desenvolvendo algoritmos de otimização (como BRKGA e ILS) e exploro o mundo da Inteligência Artificial implementando Redes Neurais do zero.

Meu objetivo é sempre buscar a intersecção entre engenharia de software e inteligência de dados para resolver problemas complexos e entregar valor real.

Automação de Consignados CAIXA

Java, HTML, CSS

Ferramenta desenvolvida para otimizar e automatizar a análise de relatórios bancários de empréstimos consignados em convênio com a Caixa Econômica Federal. O sistema processa dados brutos e facilita a validação de margens de forma rápida e visual.

Marketplace de Artesãos

Java 17, Spring Boot, Hibernate, H2 Database, Thymeleaf, HTML5, CSS3

Plataforma de e-commerce desenvolvida para a disciplina de Programação Web. O sistema atua como um marketplace que conecta artesãos (que possuem um dashboard para gerenciar suas lojas e produtos) e clientes (que podem navegar pelas vitrines, usar carrinho de compras e finalizar pedidos). O backend foi construído com a arquitetura Spring (Web, Data JPA) e renderização dinâmica via Thymeleaf.

Projetos em Sistemas Distribuídos

Java, HTML, RMI

Repositório prático focado em arquiteturas cliente-servidor e comunicação entre processos. Inclui a implementação de invocação de métodos remotos utilizando Java RMI e a estruturação de um serviço distribuído para o processamento de cálculos financeiros.

Reconhecimento de Dígitos (Rede Neural MLP)

Java

Implementação de uma Rede Neural Artificial do tipo Multi-Layer Perceptron (MLP) construída totalmente do zero. O projeto foca nos fundamentos matemáticos e lógicos da IA de baixo nível para treinar o sistema a reconhecer padrões de dígitos binários (0 e 1).

Otimização: BRKGA para Mochila Binária

Java

Aplicação de algoritmos evolutivos na área de Pesquisa Operacional. Este projeto resolve o clássico Problema da Mochila Binária utilizando a meta-heurística BRKGA (Biased Randomized-Key Genetic Algorithm), focando em maximizar valores sob restrições rígidas de capacidade.

Otimização: ILS para Caixeiro Viajante (TSP)

Java

Algoritmo desenvolvido para encontrar rotas de menor custo no Problema do Caixeiro Viajante. A solução combina uma abordagem inicial gulosa com a meta-heurística Iterated Local Search (ILS) e o refinamento de rotas via busca local 2-opt.
